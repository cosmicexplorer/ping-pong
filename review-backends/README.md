review-backends
===============

- persists *ping*s to some other format
- e.g. git-notes, github, phabricator
    - [git-appraise](https://github.com/google/git-appraise) also uses git-notes
- should cover:
  1. groups and ids (mapped to/from objects in *ping*s)
  2. publish (save)
  3. sync (refresh)
